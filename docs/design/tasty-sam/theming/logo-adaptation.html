落地规则（接 LumenXBranding.tsx）

  • 现有组件写死 src="/LumenX-cybr.png" 且 “LUMEN” 用 text-white —— 亮色下白底白字不可读，必须改造。
  • 按 presetId 切 src：暗主题 → /LumenX-cybr.png（白描边）；亮主题 → 各自亮色 PNG（均由暗色 Logo 同形重着色，透明底）。
  • teal 着色：仅 atelier-dark 加内联 style={{filter:'hue-rotate(-64deg) saturate(1.35) brightness(1.08)'}}，其余 none。
  • wordmark 文字色：全部从 text-white 改为 var(--color-text-primary)；“X” 从 #646cff 改为 var(--color-primary)。
  • atelier-light 用 logo-light-teal.png（深墨描边 + teal 核心）；brand-light 用 logo-light.png（深墨描边 + 品牌蓝核心）。二者与暗色 Logo 同形，仅换色，已替换旧莲花图 LumenX_亮色.png。
